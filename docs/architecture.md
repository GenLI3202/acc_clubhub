# ACC ClubHub 架构文档

## 系统概览

```
┌─────────────────────────────────────────────────────────┐
│                    用户访问                              │
└─────────────────────────────────────────────────────────┘
                         │
          ┌──────────────┼──────────────┐
          ▼              ▼              ▼
┌─────────────┐  ┌─────────────┐  ┌─────────────┐
│  静态网站   │  │  后端 API   │  │ Strava 同步 │
│ (Quarto)    │  │ (FastAPI)   │  │  (定时任务)  │
└─────────────┘  └─────────────┘  └─────────────┘
          │              │              │
          └──────────────┼──────────────┘
                         ▼
               ┌─────────────────┐
               │  SQLite 数据库  │
               └─────────────────┘
```

## 技术栈

- **前端**: Quarto 生成静态 HTML
- **后端**: FastAPI (Python)
- **数据库**: SQLite + SQLAlchemy
- **托管**: GitHub Pages + Render
- **CI/CD**: GitHub Actions
